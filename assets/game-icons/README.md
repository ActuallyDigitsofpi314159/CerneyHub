# Game Icons

Place custom game icons in this folder and reference them from `js/launcherData.js`.

- File path example: `assets/game-icons/mario64.svg`
- Recommended size: 256x256 or 512x512
- File format: `.svg`, `.png`, `.webp`, or `.jpg`

To update a game icon, set the game's `image` field in `js/launcherData.js` to the local asset path.
